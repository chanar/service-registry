# service-registry

## Project setup
```
npm install
```
Installs npm modules from ./package.json and ./api/package.json

### Start front-end and back-end api
```
npm run serve
cd api
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests

Front-end:
```
npm run test:unit
```

Back-end:
```
cd api
npm run test
```

### Run your end-to-end tests
Front-end:
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
